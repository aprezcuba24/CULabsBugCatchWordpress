== Description ==

This plugin let you catch and report errors to bugcatches.com so you can get a notification to your email and track all error in a single place.


== Installation ==

1. Upload `bugcatches.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the plugin settings and set the application key (you can get it from bugcatches.com)

== What is my application id ==

The application key is the id related to your application in bugcatches.com
